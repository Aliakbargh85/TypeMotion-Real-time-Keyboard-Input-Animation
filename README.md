# TypeMotion-Real-time-Keyboard-Input-Animation
A lightweight interactive keyboard visualizer that listens to real-time key presses and displays animated feedback for each key. Every pressed key triggers a visual animation and instantly appears on the screen, simulating a physical typing experience.

Press any key and watch it come to life.

## Features
- Real-time key press detection
- Animated visual feedback for each key
- Displays typed characters instantly on screen

## Tech Stack
- HTML
- CSS (Animations)
- JavaScript (Keyboard Events, DOM Manipulation)
